How to install

  1. Tap Download the app above. When the file finishes, tap Open.
  2. Android will ask if you want to allow installs from this source. Tap Settings, turn on Allow from this source, then go back.
  3. Tap Install. If Play Protect shows a warning about an unknown app, choose Install anyway (or More details then Install anyway).
  4. Open SnapRescue and point it at your unzipped Snapchat data export.
